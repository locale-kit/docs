> **A simple installation and configuration guide for LocaleKit.**

## Installation

Installation is fairly straightforward. There's no code generation magic abstracting things away - it's
as simple as using your favourite package manager to add the library to your project.

<Tabs>
<TabItem label="npm">
```sh
npx jsr add @locale-kit/locale-kit
```
</TabItem>

<TabItem label="pnpm">
```sh
pnpm dlx jsr add @locale-kit/locale-kit
```
</TabItem>

<TabItem label="Yarn">
```sh
yarn dlx jsr add @locale-kit/locale-kit
```
</TabItem>
<TabItem label="Deno">
```sh
deno add @locale-kit/locale-kit
```
</TabItem>
<TabItem label="Bun">
```sh
bunx jsr add @locale-kit/locale-kit
```
</TabItem>
</Tabs>
